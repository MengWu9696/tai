使用方法

  • 1. 請以全半形空白、全半形逗點分隔多個關鍵字。

API介紹：

本中心提供以API方式進行資安通報，分為事件通報及通報進度查詢兩種功能，以利大量及快速進行通報並掌握通報進度。本API之回傳結果皆以JSON格式呈現。

金鑰申請方式：

使用API進行資安通報或查詢通報進度，必須在API參數內使用金鑰以有效操作API，若欲申請金鑰，請先來信至twcert@cert.org.tw，主旨為「申請資安通報API金鑰-您的姓名」，例如「申請資安通報API金鑰-王大明」，並提供姓名、單位及可連絡Email等資訊。本中心在收到您的申請資料後會進行審核，通過後會提供一組金鑰給您，請注意不要任意將此金鑰外流，若偵測到異常連線，本中心有權停止您對該組金鑰的使用權。

API Request

URL: https://twcert.org.tw/api/report.php
METHOD: POST

參數 參數說明 選擇性 資料型態
key TWCERT/CC發給您的金鑰 必填 String
contact_pu 通報人（機關） 必填 String
phone 聯絡電話 必填 String
email 電子郵件 必填 String
ip IP位置
Ex: [{ "ip": "123.123.123.123", "port": "80", "port_description": "web host" }]
  • ip => 必填(ip)
  • port => 必填(port)
  • port_description => 必填(port說明)
※如果無值，可給空陣列，不知道Port請給0。 		必填 Array
web_url 網際網路位置
Ex: ["https://twcert.org.tw", "https://www.twcert.org.tw"]
※如果無值，可給空陣列。 		必填 Array
discover_date 發生時間
Ex: "1995-05-15 23:15:15" 		選填 String
model 設備廠牌、型號 選填 String
fire_wall 防火牆 選填 String
anti_virus 防毒軟體 選填 String
detected_system 入侵偵測系統 選填 String
defend_system 入侵防禦系統 選填 String
others 其他 選填 String
confidential_impact 機密性衝擊
Ex: 4, 代碼請參考事件風險等級參考表。 		必填 Integer
integrity_impact 完整性衝擊
Ex: 4, 代碼請參考事件風險等級參考表。 		必填 Integer
usability_impact 可用性衝擊
Ex: 4, 代碼請參考事件風險等級參考表。 		必填 Integer
cross_unit_impact 跨單位衝擊
Ex: 4, 代碼請參考事件風險等級參考表。 		必填 Integer
event_classification 事件分類
Ex: "A01", 代碼請參考分類對照表。 		必填 String
event_description 事件說明 必填 String
contingency_measures 應變措施 必填 String

API Response

{
　"status": "success | failure",
　"no": "您的事件編號",
　"error-codes": "description"
}
